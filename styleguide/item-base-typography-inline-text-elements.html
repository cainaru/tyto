1.2.4 Base.typography.inline-text-elements Inline text elements

Toggle example guides Toggle HTML markup

Example

This line has a link here.

This line rendered as bold text, and indicates importance.

This line rendered as italicized text, and indicates emphasis.

The b element does not have any special importance, particularly for screen readers.

The i element does not have any special importance, particularly for screen readers.

This line of text will render as underlined

This line of text is meant to be treated as deleted text.

This line of text is meant to be treated as an addition to the document.

This line of text is meant to be treated as no longer accurate.

This line of text has a word that uses superscript.

This line of text has a word that uses subscript.

This line of text is meant to be treated as fine print.

This line of text has an abbreviated word CSS.

This line of text has some keyboard input cd and ctrl + ,.

This line of text is a short inline quotation

This line of text is a citation.

The dfn element indicates a definition.

You can use the mark tag to highlight text.

For example, <section> should be wrapped as inline code.

This text is meant to be treated as sample output from a computer program.

This line of text has an equation with variables y = mx + b
