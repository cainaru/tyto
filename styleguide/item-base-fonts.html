1.1 Base.fonts Fonts

There are two variables for fonts, $sans-serif and $monospace. The fonts used for each of those variables use the "native font stack" adopted by Bootstrap v4. Read more about native font stacks in this Smashing Magazine article.

To change the font families, replace the values of the variables.
